# 实验内容  
第一部分  
生成棋盘格和chirp测试图；直接下采样，观察混叠；加高斯滤波再下采样，对比差异；画FF频谱，在频域上确认混叠消灭了  
第二部分  
验证σ公式；固定M=4，分别用 σ=0.5,1.0,1.8,4.0做实验；观察σ太小(混叠残留)和σ太大(过度模糊)的效果；和理论值σ=0.45*M=1.8对比  
第三部分  
自适应下采样:用梯度分析估计局部M值；对不同区域用不同σ滤波；和全图统一下采样对比误差图  

### 第一部分：观察混叠与抗混叠及频域分析(以chirp图为例)  
- Direct Downsample (Aliasing)（直接下采样）： 从下采样图就可以看出采样后出现了许多的鬼影，FFT图后可以看出来周围首先是不太平滑不像是一个比较正常的圆，而且图看起来很脏
- Filtered Downsample (Anti-aliasing)（高斯滤波后下采样）：像是做完了磨皮，鬼影是少了但是也丢失了斌较多的细节，不过FFT图看着舒服很多，没有了鬼影，说明高斯模糊具有一定抗混叠能力
- Original（原图）： 原图的FFT图看着就很亮很舒服，一点也不脏，说明原图的细节很丰富也没有鬼影，能量很平均分散在高频和低频
### 第二部分：验证 $\sigma$ 公式
- $\sigma = 0.5$ (Under-smoothed / 欠平滑)： 模糊力度太弱，还有许多鬼影存在，跟直接下采样没什么特别大的区别
- $\sigma = 1.0$： 混叠情况优于$\sigma = 0.5$的时候，但是在几个圆处还是有四个半圆鬼影
- $\sigma = 1.8$ (Optimal / 最优值)： 通过公式算出的最优值。几乎没有混叠，不论是采样图和FFT图都是最灰，最平均的图
- $\sigma = 2.0$ 与最优值差别不大，但是可以看得出最优值的FFT图更灰一些，更加接近原图的能量分布
- $\sigma = 4.0$ (Over-smoothed / 过度平滑)： 模糊力度太大。不仅过滤了高频，也过滤到了部分的中低频，就连FFT图都出现了一些模糊的感觉
### 第三部分：自适应下采样  
- 棋盘图和Chirp图利用自适应下采样的结果截然相反，很显然棋盘图有高频和有平滑的地方，而且间隔很大，但是Chirp图则高频低频之间没有明显的分界线，所以算法找不到各自的舒适区，跟直接用一个固定参数也没什么区别
