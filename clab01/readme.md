# 1.功能实现  
- 使用 OpenCV 读取一张测试图片  
- 输出图像基本信息，在终端打印：图像尺寸 图像通道数，像数据类型例如：宽度、长度  
- 显示原图，用 Matplotlib 或 OpenCV 显示原图  
- 转换为灰度图，将彩色图转换为灰度图，并显示  
- 保存处理结果，把灰度图保存为新文件  
- 用 NumPy 做一个简单操作，例如输出某个像素值，或者把图像左上角一块区域裁剪出来保存
