# 实验要求
已知一张包含矩形、圆、平行线、垂直线的测试图，分别对其施加：相似变换、仿射变换、透视变换，观察并总结每类变换对以下几何性质的影响
1.直线是否保持为直线
2.平行线是否仍保持平行
3.两条垂直线是否仍垂直
4.圆是否仍保持为圆
最后，对一张存在逃视略变的平面图像进行校正(自己拍一张放在桌上的带文字和表格的A4纸)，看看自己是否满意，矫正的文字是否变形
可以允许使用cv2.warpAffine、cv2.warpPergRective
必须自己做：自己构造变柜阵、自己设计测试图、自己做几何性质判断、自己写结论

用表格表示各种变换及其几何关系如下：
- 几何性质	相似变换 (Similarity)	仿射变换 (Affine)	透视变换 (Perspective)
- 直线性	✅ 直线仍保持为直线	✅ 直线仍保持为直线	✅ 直线仍保持为直线
- 平行性	✅ 平行线始终保持平行	✅ 平行线始终保持平行	❌ 平行线不再保持（相交于消失点）
- 垂直性	✅ 垂直线始终保持垂直	❌ 垂直线不再保持垂直（角度发生变化）	❌ 垂直线不再保持垂直（角度发生变化）
- 圆形保持性	✅ 圆仍保持为圆（仅半径大小改变）	❌ 圆不再保持为圆（转化为椭圆）	❌ 圆不再保持为圆（转化为椭圆或圆锥曲线）

- 矫正出来的图片，不论是图片本身还是文字都没有发生变形了
