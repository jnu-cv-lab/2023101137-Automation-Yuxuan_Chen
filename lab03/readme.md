# 一、实验内容与要求  
#### 1.图像读入与预处理  
#### 2.下采样  
将原图缩小为原来的 1/2或1/4。缩小时建议分别测试:不做预滤波直接缩小，先做高斯平滑再缩小  
#### 3.图像恢复  
将缩小后的图像恢复到原始尺寸，分别使用以下方法:最近邻内插，双线性内插，双三次内插  
#### 4.空间域比较  
对原图与恢复图进行比较，至少完成:显示原图、缩小图、恢复图，计算MSE，计算PSNR  
#### 5.傅里叶变换分析  
分别对下列图像计算二维傅里叶变换并显示频谱:原图，缩小后图像，用双线性恢复后的图像  
要求:将频谱中心移动到图像中心，对幅度谱取对数显示，比较它们的高频成分差异，并说明原因  
#### 6.DCT 分析  
分别对原图和恢复图做二维 DCT:显示 DCT系数图(可取对数显示)，统计左上角低频区域能量占总能量的比例，比较不同恢复方法下 DCT 能量分布差是并给出解释  
# 二、实验结果分析  
- 最近邻插值 NEAREST   	MSE=5.2589    PSNR=33.1677 dB
- 双线性插值 LINEAR		MSE=4.5250    PSNR=38.1481 dB
- 双三次插值 CUBIC		MSE=2.9536    PSNR=41.7190 dB
从mse/psnr知，恢复的效果CUBIC>LINEAR>NEAREST  
与原图相比，下采样图的幅度谱的四周变暗，而LINEAR恢复图的幅度谱的四周变暗得更加明显，说明经过LINEAR后高频信息丢失而恢复后也没办法恢复原来的高频细节，LINEAR回复后不看起来平滑，但没有真细节
- 原图：  		0.9986
- 最近邻：		0.9984
- 双线性：		0.9996
- 双三次： 	0.9993
原图及各恢复图像的低频能量占总能量的比例均十分接近 1，这表明图像的能量主要集中在低频区域，意味着图像整体较为平滑，细节信息匮乏、纹理特征不明显。这种能量分布特性使得图像具有良好的可压缩性，类似 JPEG 等压缩算法可舍弃更多高频系数，且不会导致图像出现明显的视觉失真。
NEAREST恢复图比原图的DCT图暗得更快，说明NEAREST恢复图对比原图，高频成分更少，图像更平滑（也可以说是模糊）、可压缩性好
